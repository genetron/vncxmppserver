Development Versions

Development versions include more features but may be unstable, so they're not intended for production use. However, current Win32 development version, 1.3.8, is a Release Candidate and it will become next stable version with no major changes. It features enhanced user interface, built-in file transfers, single-window sharing on the server side, and a greatly improved viewer. See this document for more information about new features in each particular versions.
